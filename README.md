# 🚀 Automação de Login com Playwright
Projeto de automação
## 📌 Sobre o projeto

Este projeto realiza uma automação de login em um site de teste utilizando Python e Playwright.

## 💻 Tecnologias utilizadas

* Python
* Playwright

## ⚙️ Funcionalidades

* Acessa um site automaticamente
* Preenche usuário e senha
* Realiza login automático

## ▶️ Como executar

1. Instale as dependências:
   pip install playwright

2. Instale os navegadores:
   playwright install

3. Execute o projeto:
   python main.py

## 🎯 Objetivo

Projeto criado para praticar automação web e interação com páginas.
